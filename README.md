## 💻 Pré-requisitos

Antes de começar, verifique se você atendeu aos seguintes requisitos:
* Você instalou a versão mais recente de `Python 3`
* Você tem o módulo pytube instalado.

## ☕ Usando ytDownloader

Para usar ytDownloader, siga estas etapas:

```
1. Clone o repositório em uma unidade ou pasta de sua preferência
2. Abra o promt de comando de sua preferência e digite: python main.py
3. Digite a url do vídeo que deseja fazer download
4. Abra a pasta Downloads na raiz do projeto e confira se o arquivo está lá
```
